# 🛒 Product Order Management System

## 📌 Project Overview
The Product Order Management System is a full-stack web application developed using Java and Spring Boot.
It allows users to browse products, add them to cart, and place orders, while admins can manage products and orders efficiently.

---

## 🚀 Features
- 👤 User & Admin roles
- 🛒 Add to Cart functionality
- 📦 Order Management
- 🗑️ Delete Orders
- 🔐 Secure backend logic
- 💡 Real-world business workflow implementation

---

## 🛠️ Tech Stack
- **Backend:** Java, Spring Boot
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Tools:** Maven
